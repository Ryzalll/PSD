Modeling#

from sklearn.neighbors import KNeighborsRegressor

# 3) Model KNN Regressor
knn = KNeighborsRegressor(
    n_neighbors=3,       
    weights='distance',  
    metric='minkowski',  
)

knn.fit(X_train, y_train)

Kode tersebut membuat model KNN Regressor yang akan memprediksi nilai kontinu berdasarkan 3 tetangga terdekat, di mana kontribusi tiap tetangga diberi bobot sesuai jaraknya (semakin dekat, bobot makin besar). Jarak antar sampel dihitung dengan Minkowski metric

\[ D(x, y) = \left( \sum_{i=1}^{n} |x_i - y_i|^p \right)^{\frac{1}{p}} \]

Keterangan:

  • (D(x, y)) = jarak Minkowski antara dua titik (x) dan (y)

  • (p) = parameter orde jarak (misalnya (p=1) untuk Manhattan, (p=2) untuk Euclidean)
